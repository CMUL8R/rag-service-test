# Быстрый старт RAG-сервиса

1. Установите Docker и скопируйте `.env.example` в `.env`.
2. Запустите команду `docker compose up --build`, чтобы поднять API, PostgreSQL, Redis и Qdrant.
3. Перейдите на `http://localhost:8000/static/index.html`, чтобы задать вопрос или загрузить документ.

Компоненты архитектуры:
- FastAPI отвечает за REST API и веб-интерфейс.
- PostgreSQL хранит историю запросов, включая токены и источники.
- Redis кэширует популярные ответы на час.
- Qdrant обеспечивает векторный поиск по чанкам документов.
